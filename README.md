📄 TextBin – Secure & Instant Text Sharing for Teams
TextBin is a lightweight, React-based web app that allows developers and teams to quickly share plain text or code snippets using a unique 6-digit share code. It's ideal for remote collaboration, debugging, and temporary text storage.

✨ Features:
🔐 Secure Share Codes: Easily generate a one-time share code to distribute text safely.

📥 Text Retrieval: Retrieve shared content instantly using the generated code.

📋 Clipboard Copy: Quickly copy retrieved text with a single click.

🧼 Reset on Tab Switch: Automatically clears inputs when switching tabs for better UX.

🌐 MongoDB Atlas Function API: Integrated with MongoDB Realm to store and fetch shared data.

🛠 Built With:
React.js (Functional components with Hooks)

CSS Modules (for styling)

MongoDB Atlas (Data persistence via serverless functions)
